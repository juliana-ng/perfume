# ss25-perfume-trend

## Abstract
The perfume market is projected to have the fastest growth of 6% by 2028, despite being the smallest category in the beauty industry. This growth can be explained by social media marketing on popular platforms like TikTok where there is increased spending on fragrances by young consumers who got “influenced”. TikTok is a community-focused and driven platform with engaged users and abundant insights can be gained from the comment section. This research takes a deep dive into the comments from several TikTok videos using Natural Language Processing methods like Latent Dirichlet Allocation, Named Entity Extraction (NER) and building chatbots using generative Large Language Models (LLM). The goal is to identify all-time popular perfumes and popular notes for the spring and summer seasons, as well as a chatbot to answer users’ queries using the dataset as the  context.

Keywords: Fragrance Trends, Spring and Summer Perfume, Artificial Intelligence, Natural Language Processing, Latent Dirichlet Allocation, Topic Modelling, Chatbot, Large Language Model

## Methods
1. Raw like counts
2. TF-IDF
3. LDA
4. NER
5. Chatbot
